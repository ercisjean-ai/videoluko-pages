# VideoLuko public pages

This folder contains a minimal static site for TikTok Developer setup.

## Publish on GitHub Pages

1. Create a small GitHub repository for these files.
2. Copy the contents of this folder to the root of that repository.
3. Push to the `main` branch.
4. In GitHub, open `Settings` > `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select branch `main` and folder `/root`, then save.
7. Your public URLs will be:
   - `https://YOUR-USERNAME.github.io/YOUR-REPO/`
   - `https://YOUR-USERNAME.github.io/YOUR-REPO/terms.html`
   - `https://YOUR-USERNAME.github.io/YOUR-REPO/privacy.html`
